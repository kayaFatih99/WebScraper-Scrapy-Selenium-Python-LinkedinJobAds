Escribirás clean code para desarrollar componentes de software funcionales. Expresarás tus sólidas habilidades en JavaScript, HTML, CSS3 y marcos web modernos.
Seguirás las mejores prácticas en un entorno de programación ágil, convenciones de desarrollo y una cultura DevOps.
